STEP 1: Buka Vercel
====================
https://vercel.com/new


STEP 2: Import Repository
==========================
1. Klik "Import Git Repository"
2. Pilih: hhayu8445-code/fivem-tools-v7
3. Klik "Import"


STEP 3: Configure Project
==========================
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install


STEP 4: Add Environment Variables
==================================
Klik "Environment Variables", tambahkan:

VITE_API_KEY
Value: your_base44_api_key

VITE_APP_ID
Value: your_base44_app_id

VITE_DISCORD_CLIENT_ID
Value: your_discord_client_id

VITE_DISCORD_CLIENT_SECRET
Value: your_discord_client_secret

VITE_DISCORD_REDIRECT_URI
Value: https://your-domain.vercel.app/auth/callback

VITE_DISCORD_WEBHOOK_URL
Value: your_discord_webhook_url


STEP 5: Deploy
==============
Klik "Deploy"
Tunggu 2-3 menit


STEP 6: Configure Custom Domain (Optional)
===========================================
1. Setelah deploy selesai
2. Go to: Settings ‚Üí Domains
3. Add domain: fivemtools.net
4. Update DNS:
   Type: CNAME
   Name: @
   Value: cname.vercel-dns.com


STEP 7: Update Discord OAuth
=============================
1. https://discord.com/developers/applications
2. Pilih aplikasi Anda
3. OAuth2 ‚Üí Redirects
4. Add: https://your-domain.vercel.app/auth/callback
5. Save
