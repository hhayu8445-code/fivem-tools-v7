========================================

OPTION 1: BUAT REPOSITORY BARU (RECOMMENDED)
============================================

1. Buka browser, pergi ke: https://github.com/new

2. Isi form:
   - Repository name: fivem-tools-v7
   - Description: FiveM Tools V7 - Ultimate Resource Hub
   - Public/Private: Public
   - JANGAN centang "Initialize this repository with:"
   
3. Klik "Create repository"

4. Setelah repository dibuat, jalankan commands ini:

   cd "c:\Users\MUDDING UNDERGROUND\Pictures\runkzerigala\DENGNA"
   
   git remote remove origin
   
   git remote add origin https://github.com/boostfivem4-oss/fivem-tools-v7.git
   
   git push -u origin main

5. Jika diminta login:
   - Browser akan terbuka otomatis
   - Login dengan GitHub account Anda
   - Authorize Git Credential Manager
   - Kembali ke terminal, push akan otomatis lanjut

========================================
  SETELAH PUSH BERHASIL
========================================

1. Buka: https://vercel.com/new

2. Import repository: boostfivem4-oss/fivem-tools-v7

3. Configure:
   - Framework: Vite
   - Build Command: npm run build
   - Output Directory: dist

4. Add Environment Variables:
   VITE_API_KEY=your_base44_api_key
   VITE_APP_ID=your_base44_app_id
   VITE_DISCORD_CLIENT_ID=your_discord_client_id
   VITE_DISCORD_CLIENT_SECRET=your_discord_client_secret
   VITE_DISCORD_REDIRECT_URI=https://your-domain.vercel.app/auth/callback
   VITE_DISCORD_WEBHOOK_URL=your_discord_webhook_url

5. Klik "Deploy"

6. Tunggu deploy selesai (2-3 menit)
